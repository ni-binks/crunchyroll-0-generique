# Crunchyroll 0 Générique
Saute les openings et les crédits pour passer directement à la suite et éviter les spoilers post-generique.

# Installation
- Installer un gestionnaire d'userscript comme ViolentMonkey (Alternative OpenSource de TamperMonkey).
- Ajouter le script en cliquant ici [Source GitHub](http://github.com/ni-binks/crunchyroll-0-generique/raw/refs/heads/main/Crunchyroll0Generique.user.js) ou en copiant le contenu de `Crunchyroll0Generique.user.js` dans un nouveau script.
